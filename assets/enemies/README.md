# PERSONALIZAR ENEMIGOS CON IMAGENES

## Brainrots (minions aliados)

Los Brainrots pueden usar hasta 3 imágenes diferentes.

### Como poner imagenes a los Brainrots

1. Crea 3 imágenes PNG (o menos)
2. Renombralas como:
   - `brainrot_0.png`
   - `brainrot_1.png`
   - `brainrot_2.png`
3. Colocalas en: `assets/enemies/`

Cada Brainrot elige una de las 3 imagenes al azar cuando aparece.

### Tamaño recomendado

- 24x24 pixeles (se escala automaticamente)
- PNG con canal alfa

### Que pasa si no hay imagen

Si no existen los archivos, los Brainrots se dibujan como circulos morados con ojos (como siempre).
